Описание файлов программы:
	./compile.sh - компиляция
	./run_server.sh - запуск сервера
	./stop_servr.sh - остановка сервера(требует пароль админа)
	./clear.sh - удаляет скомпилированные программы
	client - программа клиент(появляется после компиляции)
	server - программа сервер
	client.c - код
	server.c - код
	header.h - заголовочный файл

1. Для запуска *.sh должны иметь права x (chmod +x *.sh)

2. Сам сервер берет свой порт из константы в header.h

3. Сначала необходимо запустить сервер ./run_server.sh

4. Пример запуска: client -i 127.0.0.1 -p 10000 -c whoami

5. Ключи -i -p -c обязательны.

6. Если команда серверу состоит из нескольких слов то ее необходимо в ""
	client -i 127.0.0.1 -p 10000 -c "ls -la"

7. Остановить сервер можно либо передав ему команду:
	client -i 127.0.0.1 -p 10000 -c exit
   Либо:
   	./stop_server.sh
